# Agenda - CRUD

## Technologies used:
- [Webpack](https://webpack.js.org/)
- [Node.js](https://nodejs.org/en/)
- [BootStrap](https://getbootstrap.com/)
- [CSRF token](https://portswigger.net/web-security/csrf/tokens)

## Commands:

